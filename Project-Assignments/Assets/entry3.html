Week 4-7: UI/UX Design - Intro to HTML/CSS: Making Webpages

Within these weeks, I learned about HTML, the backbone of web pages. It’s fascinating to see how simple tags like "<p>" and "<h1>" can structure content.

Here’s a simple example of what I practiced:

                    <html>
                        <head>
                            <title>My First Page</title>
                        </head>
                        <body>
                            <h1>Hello, World!</h1>
                            <p>Welcome to my first webpage.</p>
                            <a href="https://example.com">Visit Example</a>
                        </body>
                    </html>

We learnt a lot about CSS too, the magic behind a beautiful webpage. CSS has been a game-changer! I learned about selectors, properties, and how to make web pages visually appealing.

Here’s an example of a CSS rule I created:

                        footer {
                            text-align: center;
                            padding: 10px 0;
                            background: #333;
                            color: white;
                        }

With CSS, I transformed my plain HTML into something visually appealing and user-friendly. Building my first webpage was exciting experience. It shows that I am capable to do whatsoever I'm determined to do!
